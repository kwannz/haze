1//! Technical Indicators Module
2//!
3//! # Overview
4//! This module provides a comprehensive suite of technical analysis indicators
5//! commonly used in quantitative trading. All indicators are implemented with
6//! O(n) time complexity and follow consistent NaN-handling conventions for
7//! warmup periods.
8//!
9//! # Available Sub-modules
10//!
11//! ## Core Indicators
12//! - [`momentum`] - RSI, MACD, Stochastic, CCI, Williams %R, ROC, MOM, etc.
13//! - [`volatility`] - ATR, Bollinger Bands, Keltner Channels, Donchian, etc.
14//! - [`trend`] - ADX, SuperTrend, PSAR, Aroon, Choppiness Index, etc.
15//! - [`volume`] - OBV, VWAP, MFI, CMF, Accumulation/Distribution, etc.
16//! - [`overlap`] - Moving averages and overlay studies
17//!
18//! ## Advanced Indicators
19//! - [`harmonics`] - Harmonic pattern detection
20//! - [`fibonacci`] - Fibonacci retracement and extension levels
21//! - [`ichimoku`] - Ichimoku Cloud components
22//! - [`pivots`] - Pivot points and support/resistance levels
23//! - [`candlestick`] - Candlestick pattern recognition
24//!
25//! ## Specialized Indicators
26//! - [`price_transform`] - Price transformation functions
27//! - [`cycle`] - Cycle analysis indicators
28//! - [`sfg`] - SFG indicator suite
29//! - [`sfg_signals`] - SFG signal generation
30//! - [`sfg_utils`] - SFG utility functions
31//! - [`pandas_ta`] - Pandas-TA compatible implementations
32//!
33//! # Usage Pattern
34//! All indicators follow a consistent API pattern:
35//! - Input: Price arrays (close, high, low, open, volume as needed)
36//! - Output: `Vec<f64>` with NaN for warmup periods
37//! - Period parameter controls lookback window
38//!
39//! # Examples
40//! ```rust,no_run
41//! use haze_library::indicators::{momentum::rsi, trend::supertrend};
42//!
43//! let close = vec![
44//!     100.0, 101.0, 102.0, 103.0, 104.0, 105.0, 104.5, 104.0, 105.0, 106.0,
45//!     107.0, 108.0, 109.0, 110.0, 111.0, 112.0, 113.0, 114.0, 115.0, 116.0,
46//! ];
47//! let high: Vec<f64> = close.iter().map(|&v| v + 1.0).collect();
48//! let low: Vec<f64> = close.iter().map(|&v| v - 1.0).collect();
49//!
50//! // Momentum indicator
51//! let rsi_values = rsi(&close, 14).unwrap();
52//! assert_eq!(rsi_values.len(), close.len());
53//!
54//! // Trend indicator with OHLC data
55//! let (st_line, direction, upper, lower) = supertrend(&high, &low, &close, 10, 3.0).unwrap();
56//! assert_eq!(st_line.len(), close.len());
57//! assert_eq!(direction.len(), close.len());
58//! assert_eq!(upper.len(), close.len());
59//! assert_eq!(lower.len(), close.len());
60//! ```
61//!

65//! - [`crate::utils::streaming`] - Online/incremental indicator versions
